# 🕸️ Web Scraping with Selenium

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python)
![Selenium](https://img.shields.io/badge/Selenium-4.x-green?logo=selenium)
![License](https://img.shields.io/badge/License-MIT-yellow)

A collection of Python scripts for scraping static and dynamic websites using **Selenium WebDriver**. Covers real-world use cases including data extraction, pagination handling, and exporting results to CSV/JSON.

---

## 📌 What This Project Does

- Automates browser interactions to extract data from websites
- Handles dynamic content rendered by JavaScript
- Navigates multi-page results with pagination
- Exports scraped data to structured formats (CSV, JSON)

---

## 🧰 Tech Stack

| Tool | Purpose |
|---|---|
| `Python 3.8+` | Core language |
| `Selenium 4.x` | Browser automation & scraping |
| `ChromeDriver` | Chrome browser driver |
| `Pandas` | Data cleaning & export |

---

---

## 📦 Requirements

```
selenium>=4.0.0
```

## ⚠️ Notes

- Always check a website's `robots.txt` before scraping
- Add delays between requests to avoid rate limiting
